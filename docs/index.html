世界杯投注分析助手

一个 Python Streamlit MVP，用于辅助分析世界杯小组赛普通胜平负、让球胜平负、EV、市场概率和出线情景。

部署说明：GitHub Pages 只能托管静态网页，不能直接运行 Streamlit 后端。因此这个页面是项目介绍页，不是完整交互版应用。

完整应用需要在本地、Streamlit Community Cloud、Render、Railway 等支持 Python 服务的平台运行。

当前能力

  • 本地 CSV 数据源和中国体彩实时数据 provider 架构。
  • 手动录入中国体彩风格赔率。
  • 普通胜平负和让球胜平负市场分析。
  • 无水概率、模型概率、edge、EV 和风险判断。
  • 本地小组匹配时显示出线情景；无法匹配时仍可做市场分析。
  • 生成中文比赛分析报告。

本地运行

python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
PYTHONPATH=src streamlit run src/app/streamlit_app.py

测试

pytest
